# Pontos de quebra CSS

existem pontos de quebras importantes, que precisam ser levados em consideração quando queremos visualizar um aplicativo web pelo celular.

    - Mobile 
        - 360 x 640 
        - 375 x 667 
        - 360 x 720
    - Iphone
        - 375 x 812

    - Tablet
        - 768 x 1024

    - Laptop
        - 1366 x 768

    Desktop pra cima
        - 1920 x 1080
